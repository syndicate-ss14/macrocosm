# Reporting a security vulnerability
You can report a security vulnerability to Wizard's Den through Discord or through email.

If you want to send an email, you can contact Wizard's Den at <support@spacestation14.com>.
If you want to contact Wizard's Den through Discord, you can join [the SS14 server](https://discord.gg/MwDDf6t)
and then **privately** message anyone with the `@Wizard` or `@SS14 Maintainer` role.

In either case, **do not publicly disclose the vulnerability until you are explicitly given permission to do so**.
